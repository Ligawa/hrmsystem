# Secure Document Portal - Quick Start Guide

## For Applicants

### How to Submit Your Documents

1. **Check Your Email**
   - Look for the "World Vision" application confirmation email
   - It will contain a link to the secure upload portal

2. **Access the Portal**
   - Click: "Access Secure Upload Portal" in the email
   - Or go to: `/application-portal` on the website

3. **Sign In**
   - Enter your email address
   - Click "Access Portal"
   - (No password needed)

4. **Submit Your Video Interview**
   - Record a 5-7 minute video answering the interview questions
   - Upload to: Loom, YouTube, Google Drive, or Vimeo
   - Paste the video link in the portal
   - Click "Add"

5. **Upload Your Documents**
   - **Drag and drop** files onto the upload area, OR
   - Click **"browse files"** to select files
   - Accepted types: PDF, images, Word documents, videos
   - Max 50MB per file

6. **Required Documents**
   - ✓ Valid government-issued ID (passport, national ID, driver's license)
   - ✓ Educational certificates/degrees
   - ✓ Professional certifications (if applicable)

7. **Review & Submit**
   - Check that all documents are uploaded
   - Ensure video link is added
   - Click "Submit Application"
   - Wait for confirmation message

8. **Confirmation**
   - You'll see: "Your application has been successfully submitted"
   - HR team will review and contact you soon

## For HR/Admin Team

### Accessing Submissions

1. **Go to Admin Dashboard**
   - URL: `/setup` (requires login)

2. **View Quick Actions**
   - Click: "View Document Submissions"
   - Or navigate to: `/setup/submissions`

3. **View Submission List**
   - See all applicant submissions in a table
   - Columns: Email, Submitted Date, Document Count, Video, Actions

4. **Review Individual Submission**
   - Click the "View" button on any row
   - Opens a detailed modal showing:
     - Applicant email (with copy button)
     - Video interview link (opens in new tab)
     - All uploaded documents with file details
     - Submission timestamp

5. **Actions Available**
   - Copy applicant email address
   - Open video interview link
   - View document details
   - Create notes (in your CRM/tracking system)

## File Upload Guide

### Supported Formats

| Type | Extensions |
|------|-----------|
| Documents | .pdf |
| Images | .jpg, .jpeg, .png |
| Word | .doc, .docx |
| Video | .mp4, .mov |

### File Size Limits

- Maximum: 50MB per file
- Multiple files supported
- No total size limit

### Tips for Best Results

1. **ID Documents**
   - Use high-quality scans
   - Ensure text is readable
   - Both sides if applicable

2. **Certificates**
   - Include official documents
   - Make sure dates are visible
   - PDF format recommended

3. **Video Interview**
   - Use Loom (easiest, recommended)
   - Ensure good lighting
   - Clear audio
   - Test link before submitting
   - 5-7 minutes recommended

## Troubleshooting

### "I can't access the portal"
**Solution:**
- Copy the exact link from email
- Try in different browser
- Check internet connection
- Email: careers@worldvision.org for help

### "File won't upload"
**Check:**
- File size is under 50MB
- File type is supported (.pdf, .jpg, .png, .doc, .docx, .mp4, .mov)
- Browser has permission to access files

### "Video link isn't working"
**Make sure:**
- Link is from: Loom, YouTube, Google Drive, or Vimeo
- Link is shareable/public
- No special characters in URL

### "I submitted but didn't get confirmation"
**This is normal:**
- Confirmation appears on screen immediately
- Check spam folder for follow-up emails
- Contact HR team if concerned

## FAQ

**Q: Can I upload documents later?**
A: Submit everything at once. You cannot add documents after submission.

**Q: What if I make a mistake?**
A: Contact careers@worldvision.org. HR can discuss resubmission if needed.

**Q: Are my documents private?**
A: Yes. Only World Vision HR team can access your materials.

**Q: Can I change my video after submission?**
A: No, but you can contact HR to discuss resubmission.

**Q: What video platform should I use?**
A: Loom is recommended (easiest), but YouTube, Google Drive, and Vimeo also work.

**Q: How big can files be?**
A: Maximum 50MB per file. No limit on number of files.

**Q: Can I submit on mobile?**
A: Yes, the portal works on phones and tablets.
